# 页面文案配置 - 访问指南

## 如何访问页面文案配置后台

页面文案配置功能现在可以通过**两种方式**访问：

---

## 方式一：通过多场景配置页面（推荐）

### 访问步骤

1. **打开多场景配置管理后台**
   ```
   https://你的域名/admin/share-config
   ```

2. **输入管理员密码**
   ```
   plantlogic2026
   ```

3. **点击右上角「页面文案配置」按钮**

   在页面顶部右上角，「刷新」按钮旁边，有一个蓝色的「页面文案配置」按钮。

4. **进入文案配置页面**

   点击后会跳转到 `/admin/page-content` 页面。

---

## 方式二：直接访问独立路由

直接在浏览器中访问：
```
https://你的域名/admin/page-content
```

这会直接打开页面文案配置界面，无需通过多场景配置页面。

---

## 方式三：通过主应用管理后台（仅限管理员）

1. **以管理员身份登录主应用**

   确保你的账号已被设置为管理员（`is_admin = true`）

2. **点击底部导航的「管理」标签**

3. **点击「页面文案配置」标签**

   在管理后台顶部会有三个标签：
   - 音频管理
   - 知识库录入
   - 页面文案配置

---

## 快速访问链接

| 入口 | URL | 说明 |
|------|-----|------|
| 多场景配置 | `/admin/share-config` | 需要密码，可跳转到文案配置 |
| 文案配置直达 | `/admin/page-content` | 直接访问文案配置 |
| 主应用管理 | `/admin` | 需要管理员权限，主应用内置 |

---

## 使用流程

访问文案配置页面后：

1. **选择场景**：在顶部输入场景标识（如 `default`、`1234test`）
2. **编辑文案**：修改各个页面的文案内容
3. **保存修改**：点击「保存所有修改」按钮
4. **验证效果**：访问引流页面查看效果

---

## 从文案配置返回多场景配置

在页面文案配置页面的右上角，有一个「返回多场景配置」按钮，点击即可返回。
