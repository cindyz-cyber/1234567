## 🛠️ 技术实现

### 数据库表结构

#### 1. `chakra_narrative_templates` - 脉轮叙事模板

```sql
CREATE TABLE chakra_narrative_templates (
  id serial PRIMARY KEY,
  center_name text NOT NULL,           -- 能量中心名称
  tier text NOT NULL,                  -- 梯度 (high/medium/low)
  keywords text[] NOT NULL,            -- 关键词库
  templates text[] NOT NULL,           -- 描述模板库
  UNIQUE(center_name, tier)
);
```

**数据覆盖:**
- 3个能量中心（松果体、心轮、喉轮）
- 每个中心3段梯度
- 共9条模板记录
- 每条3-5个变体模板

#### 2. `tone_dynamics` - 调性动态动词

```sql
CREATE TABLE tone_dynamics (
  tone_number int PRIMARY KEY,         -- 调性编号 (1-13)
  color_temperature text NOT NULL,     -- 色温 (cold/warm)
  verbs text[] NOT NULL,               -- 动词库
  adjectives text[] NOT NULL           -- 形容词库
);
```

**数据覆盖:**
- 13个调性完整覆盖
- 每个调性6-8个动词
- 每个调性6-8个形容词

#### 3. `quantum_synergy_bursts` - 量子共振爆发短句

```sql
CREATE TABLE quantum_synergy_bursts (
  synergy_code int PRIMARY KEY,        -- 共振代码
  relationship_type text NOT NULL,     -- 关系类型
  burst_templates text[] NOT NULL      -- 爆发式短句模板
);
```

**数据覆盖:**
- 5种核心共振类型
- 每种类型3-4个变体短句

#### 4. `wind_year_ambient` - 白风年环境音

```sql
CREATE TABLE wind_year_ambient (
  id serial PRIMARY KEY,
  context text NOT NULL,               -- 上下文（喉轮专属/通用）
  imagery text[] NOT NULL              -- 意象库
);
```

**数据覆盖:**
- 喉轮专属意象: 6条
- 通用环境音: 3条

### 核心函数

#### `renderComprehensiveChakraNarrative()`

**功能**: 综合渲染 - 脉轮 + 调性 + 环境音

**输入参数:**
- `centerName`: 能量中心名称
- `percentage`: 分值（0-100）
- `toneNumber`: 调性编号（1-13）
- `includeWindYear`: 是否包含白风年环境音

**输出结构:**
```typescript
{
  narrative: string;        // 完整叙事文本
  tier: string;            // 梯度 (high/medium/low)
  keywords: string[];      // 关键词数组
  toneModifier?: string;   // 调性修饰词
  windYearImagery?: string; // 白风年意象（如果触发）
}
```

**渲染逻辑:**
```typescript
1. 判断梯度 (percentage → tier)
2. 从数据库获取对应模板
3. 随机选择模板（避免重复）
4. 替换变量 {percentage}
5. 融入调性动词（冷/暖色调）
6. 20-30%概率添加白风年环境音（仅喉轮）
7. 返回完整叙事
```

#### `renderQuantumSynergyBurst()`

**功能**: 渲染量子共振爆发短句

**输入参数:**
- `synergyCode`: 共振代码（如 261, 130）

**输出:**
- 爆发式短句字符串

**应用场景:**
- 关系分析报告
- 量子共振检测
- 能量场干涉描述
