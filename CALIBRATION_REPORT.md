### 技术实现

系统采用**多锚点校准算法**：
- 使用三个校准点作为绝对参考
- 计算任意日期时，自动选择最近的校准点
- 从该锚点进行天数偏移计算
- 确保在260天卓尔金周期内正确环绕

  - 互动：太阳之光穿透深海黑暗，照亮意识探索之路

  - 互动：太阳之母孕育星网，激活和谐共振天赋

### 2. 子时双印记逻辑

**触发条件：**
- 用户勾选"子时出生"（00:00 - 01:00）

**计算逻辑：**
1. 计算出生当日的 Kin（主印记）
2. 计算次日的 Kin（次印记）
3. 分别计算两个印记的松果体能量值
4. **取两者松果体最高值**作为最终松果体能量
5. 双印记能量叠加到喉轮和心轮

**示例：**
```
出生日期：2023-02-10 子时
- 主印记：Kin 8（银河黄星星）
- 次印记：Kin 9（太阳红月）
- 松果体能量 = max(Kin8松果体, Kin9松果体)
- 状态标记："子时双印记"
```
