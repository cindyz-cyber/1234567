# 脉轮共振感应渲染系统更新报告

## 更新概述

成功实现了基于能量分值的"感应渲染"系统，为每个图腾的能量中心提供动态的、具有情感共鸣的描述。

## 核心特性

### 1. 高分/低分双模描述体系

- **高分模式（≥70%）**: 能量中心开放，直觉主导，展现图腾的核心天赋
- **低分模式（<70%）**: 能量中心防御，逻辑主导，展现保护性特质

### 2. 四大色系家族特质

#### 红色家族（生存与滋养）
- 红龙、红蛇、红月、红天行者、红地球
- 核心主题：原始生命力、母体能量、空间穿越

#### 白色家族（净化与跨越）
- 白风、白世界桥、白狗、白巫师、白镜
- 核心主题：灵性传播、连接维度、真理映照

#### 蓝色家族（蜕变与逻辑）
- 蓝夜、蓝手、蓝猴、蓝鹰、蓝风暴
- 核心主题：深层直觉、知行合一、破局变革

#### 黄色家族（开悟与统筹）
- 黄种子、黄星星、黄人、黄战士、黄太阳
- 核心主题：觉知成长、自由意志、普照统筹

## 技术实现

### 1. 数据库表结构

```sql
CREATE TABLE chakra_resonance_descriptions (
  totem_id int PRIMARY KEY,
  totem_name_cn text,
  color_family text,
  high_score_description text,
  low_score_description text,
  created_at timestamptz
);
```

### 2. 核心函数

#### `renderChakraResonance(kin, percentage, centerName)`
根据 Kin 和分值动态渲染感应描述

#### `renderMultipleChakraResonances(kin, centers)`
批量渲染多个能量中心

#### `getTotemCoreEssence(kin)`
获取图腾的核心特质（高分模式）

### 3. 渲染逻辑

```typescript
// 自动判断高分/低分
const isHighScore = percentage >= 70;

// 从数据库获取对应描述
const description = isHighScore
  ? data.high_score_description
  : data.low_score_description;

// 生成完整描述
return `${centerName} ${percentage}% - ${description}`;
```
