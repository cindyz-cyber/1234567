## 数据库字段说明

### h5_share_config 表

| 字段 | 类型 | 说明 |
|------|------|------|
| scene_token | text | 场景标识（如 journeyzen2026） |
| scene_name | text | 场景名称（如"禅修之旅"） |
| is_active | boolean | 是否启用（true=显示，false=隐藏） |
| daily_token | text | 访问令牌 |

### page_visibility_config 表

| 字段 | 类型 | 说明 |
|------|------|------|
| scene_token | text | 场景标识 |
| page_name | text | 页面名称（naming, home, emotion等） |
| is_visible | boolean | 是否显示（true=显示，false=跳过） |

## 常见问题

### Q1: 修改后台开关后，前台还是显示旧状态？

**原因**：浏览器标签页已经加载过一次，需要刷新

**解决**：点击「刷新重试」按钮或按 F5 刷新页面

### Q2: 子页面隐藏后，流程直接跳到卡片页？

**原因**：可能中间所有页面都被隐藏了

**解决**：至少保留一个中间页面可见（建议保留日记页）

### Q3: 控制台报错"场景不存在"？

**原因**：scene_token 拼写错误或未配置

**解决**：
1. 检查 URL 中的 `?scene=xxx` 参数是否正确
2. 到后台 `/admin/share-config` 创建对应场景
