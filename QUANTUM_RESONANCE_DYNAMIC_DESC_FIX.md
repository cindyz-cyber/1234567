## 🔧 技术细节

### 动态算法原理

**分数生成**:
```typescript
const dynamicOffset = (kinA + kinB) % 29;
synergyScore = 65 + dynamicOffset;
```

**为什么是 29？**
- 29 是玛雅历法中的「月亮循环」基数
- 确保分数范围在 65% ~ 93% 之间（合理的共振区间）
- 避免极端值（如 0% 或 100%，保留给特殊关系）

**为什么从 65% 起始？**
- < 65%: 能量场过于分散，难以产生有效协同
- 65% - 75%: 需要主动磨合的差异化连接
- 75% - 85%: 自然舒适的温和共振
- 85% +: 高度契合的协同场域
